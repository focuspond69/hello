Project Overview

Build an interactive quiz application that allows users to test their knowledge on various topics. The app will feature multiple-choice questions, a timer, score tracking, and immediate feedback on answers.

Key Features:

  • Multiple-choice questions with immediate feedback
  • Timer for each question
  • Score tracking and final results
  • Progress indicator
  • Random question selection
  • Responsive design for all devices
  • Local storage for high scores

Project Requirements
