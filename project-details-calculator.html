Project Overview

Build a simple calculator that can perform basic arithmetic operations. This project will help you understand DOM manipulation, event handling, and basic JavaScript operations.

Key Features:

  • Basic arithmetic operations (addition, subtraction, multiplication, division)
  • Clear and delete functionality
  • Decimal point support
  • Keyboard input support
  • Responsive design
  • Error handling for invalid operations

Project Requirements

Prerequisites:

  • Basic knowledge of HTML and CSS
  • Understanding of JavaScript fundamentals
  • Familiarity with DOM manipulation
  • Basic understanding of event handling
